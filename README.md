### MOD 简介

添加了BangDream企划中，Ave Mujica 乐队所属的角色——丰川祥子。

祥子拥有75张标准卡牌，以及独有的，存放于额外卡堆的音乐牌32张。

本MOD添加了5个新事件、一个终幕boss以及10+专属遗物。

### 机制介绍

祥子可以在精英战与Boss战中收集音乐牌。音乐牌在战斗中存放在独立卡堆“歌单”中，通过卡牌效果可以从歌单中抽牌，达成条件即可触发演奏获得特殊效果。

另外，祥子在普通战斗中，也可能遭遇到意想不到的事件。
